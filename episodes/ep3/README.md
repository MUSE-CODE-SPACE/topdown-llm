# Top-Down LLM — EP3 코드 (스냅샷)

이 폴더 = EP3 시점의 전체 코드예요. **Git 없이도** 이 폴더만 받아서 쓰면 됩니다.

## 실행
```bash
npm install
cp .env.example .env.local   # .env.local 에 본인 Anthropic 키 붙여넣기
npm run dev                  # http://localhost:3000
```
키 발급: https://console.anthropic.com
